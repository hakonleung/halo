# 工作流指令

执行 AI Coding 工作流，管理需求开发全流程。

## 触发指令

```
/flow-start [需求]
```

初始化流程：生成单号 `PRD_XXX`，创建分支，创建目录，开始执行

## 重要提示

**在执行此命令前，必须先读取完整的详细说明文件：**

`ai-coding/action-work.md`

该文件包含完整的执行流程、详细步骤、输出格式等所有必要信息。请先读取该文件，然后按照其中的详细说明执行。

## 执行流程

1. 创建分支 `feat/PRD_XXX`
2. 创建工作目录
3. 初始化 overview.json
4. 执行开发阶段（01-07）
5. 每阶段 AI 执行 → AI 自验收 → 人类验收 → Git Commit

## 参数说明

- `[需求]`: 需求描述，用于生成工作流单号和标题

## 详细说明

完整的工作流说明、执行步骤、输出格式等详细信息，请参考：

`ai-coding/action-work.md`

**请务必在执行前阅读该文件的完整内容。**
