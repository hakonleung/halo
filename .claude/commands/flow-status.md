# 查看工作流状态

查看当前工作流的执行状态。

## 执行步骤

1. 读取 `ai-works/` 目录，找到最近活跃的工作流
2. 读取对应的 `overview.json` 文件
3. 输出状态摘要

## 输出格式

```
工作流: [单号]
分支: [Git 分支名]
状态: [进行中/暂停/已完成]
当前阶段: [阶段名]
进度: [已完成阶段数]/[总阶段数]

已完成阶段:
- [x] 01-需求澄清
- [x] 02-PRD
- [ ] 03-UI设计 (当前)
- [ ] ...

最后更新: [时间]
```

## 如果没有活跃工作流

提示用户使用 `/flow-start` 或 `/flow-quick` 开始新的工作流。

请执行状态检查。
