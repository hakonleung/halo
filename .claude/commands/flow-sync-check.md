# 检查文档代码同步

检查设计文档与代码实现的同步状态。

## 参数

$ARGUMENTS

格式: `[单号]?`（可选，不提供则检查最近的工作流）

## 检查矩阵

| 文档 | 检查项 | 代码对应 |
|------|--------|---------|
| api-spec.md | API 端点 | `src/app/api/**` |
| tech-design.md | 数据模型 | `src/db/schema.ts` |
| ui-design.md | 组件清单 | `src/components/**` |

## 执行步骤

1. 确定工作流单号
2. 读取设计文档，提取：
   - API 端点列表
   - 数据模型字段
   - UI 组件清单
3. 扫描代码，检查对应实现
4. 计算同步率
5. 输出报告

## 输出格式

```
同步检查报告 - [单号]
================

API 同步率: 95% (19/20)
- [x] GET /api/users
- [x] POST /api/users
- [ ] DELETE /api/users/:id (未实现)

数据模型同步率: 100% (15/15)
- [x] User.id
- [x] User.email
...

组件同步率: 90% (18/20)
- [x] LoginForm
- [ ] PasswordResetForm (未实现)
...

总体同步率: 95%
```

## 同步率门槛

| 阶段 | 最低同步率 |
|------|----------|
| 05 完成 | ≥ 90% |
| 06 验证 | ≥ 95% |
| 07 部署 | = 100% |

请执行同步检查。
