# 列出所有工作流

列出所有工作流及其状态。

## 执行步骤

1. 扫描 `ai-coding/works/` 目录下的所有子目录
2. 读取每个工作流的 `overview.json`
3. 按最后更新时间倒序排列
4. 输出列表

## 输出格式

```
| 单号 | 类型 | 状态 | 当前阶段 | 最后更新 |
|------|------|------|---------|---------|
| PRD_001 | 完整 | 已完成 | 07-部署 | 2026-01-20 |
| FIX_002 | 轻量 | 进行中 | Q2-实现 | 2026-01-25 |
| ITR_001 | 迭代 | 暂停 | I2-设计 | 2026-01-23 |
```

## 类型说明

- `PRD_XXX` - 完整工作流
- `FIX_XXX` / `FEAT_XXX` / `OPT_XXX` / `UI_XXX` - 轻量工作流
- `ITR_XXX` - 迭代工作流
- `REF_XXX` - 重构工作流

请执行列表查询。
