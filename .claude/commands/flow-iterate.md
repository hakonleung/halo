# 迭代工作流指令

在已完成的 PRD 基础上进行功能迭代，复用已有设计文档。

## 触发指令

```
/flow-iterate [PRD_XXX] [迭代需求]
```

**示例**: `/flow-iterate PRD_001 给行为记录列表增加搜索和筛选功能`

## 重要提示

**在执行此命令前，必须先读取完整的详细说明文件：**

`ai-coding/action-iterate.md`

该文件包含完整的执行流程、详细步骤、输出格式等所有必要信息。请先读取该文件，然后按照其中的详细说明执行。

## 执行流程

1. 加载原 PRD 上下文
2. I1: 增量需求分析
3. I2: 增量设计
4. I3: 增量实现
5. I4: 回归验证
6. Git Commit

## 参数说明

- `[PRD_XXX]`: 原 PRD 单号
- `[迭代需求]`: 迭代需求描述

## 适用场景

- 功能增强
- 需求变更
- 版本迭代
- 补充功能
