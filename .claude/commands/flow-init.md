# 初始化项目指令

创建或初始化 AI Coding 工作流配置。

## 触发指令

```
/flow-init
```

## 重要提示

**在执行此命令前，必须先读取完整的详细说明文件：**

`ai-coding/action-init.md`

该文件包含完整的执行流程、详细步骤、输出格式等所有必要信息。请先读取该文件，然后按照其中的详细说明执行。

## 执行流程

1. 询问是否从现有项目初始化
2. 阶段 1: 生成 project.md
3. 阶段 2: 生成 ui-config.md
4. 阶段 3: 生成 tech-config.md
5. 完成

## 参数说明

无参数

## 详细说明

完整的工作流说明、执行步骤、输出格式等详细信息，请参考：

`ai-coding/action-init.md`

**请务必在执行前阅读该文件的完整内容。**
