# UI 配置

> 项目特定的 UI 配置信息，供 03-UI设计 阶段引用。

## 品牌

- **名称**: NEO-LOG
- **理念**: AI 原生的个人生活追踪系统，赛博朋克美学与实用主义结合
- **Slogan**: Log your life. Hack your future.
- **风格**: 赛博朋克 + 复古未来主义 + Matrix 数据流 + 科幻 HUD

## 配色

- **荧光白** (`#E0E0E0`): 主文字

- **成功**: `#00FF41`
- **警告**: `#FFD700`

## 组件规范

### 通用

- **默认圆角**: 4px
- **边框样式**: 1px solid, 带故障动画
- **故障动画**: 随机2-5秒触发，闪烁/位移50-100ms

### 卡片

```
背景: #1A1A1A
边框: 1px solid rgba(0, 255, 65, 0.3)
发光: box-shadow: 0 0 15px rgba(0, 255, 65, 0.1)
悬停: 边框亮度+，发光半径+
```

### 按钮

**Primary**: 渐变背景(#00FF41→#00CC33)，文字#0A0A0A，悬停亮度+10%
**Secondary**: 透明背景，矩阵绿文字/边框，悬停填充rgba(0,255,65,0.1)
**Danger**: 透明背景，霓虹红(#FF3366)文字/边框

### 输入框

```
背景: #0A0A0A
边框: 1px solid #2A2A2A
聚焦: 边框#00FF41 + 外发光
光标: 闪烁块状（可选）
占位符: #555555
```

### 标签/徽章

```
背景: 半透明主色 rgba(0, 255, 65, 0.2)
文字: 对应主色
边框: 1px solid 对应主色
字体: Micro尺寸，全大写
```

## 动画

### 过渡时长

| 场景 | 时长 | 缓动 |
|------|------|------|
| 悬停 | 150ms | ease-out |
| 页面切换 | 300ms | ease-in-out |
| 弹窗出现 | 200ms | ease-out |
| 弹窗消失 | 150ms | ease-in |

### 特色动画

- **故障(Glitch)**: 触发场景: 按钮点击、错误提示、加载完成, 时长: 100-200ms
- **打字机**: 触发场景: AI回复、重要文字, 时长: 30-50ms/字符
- **矩阵雨**: 触发场景: 加载状态、AI思考、背景, 时长: 持续
- **扫描线**: 触发场景: 页面背景、卡片装饰, 时长: 3-5s周期
- **脉冲发光**: 触发场景: 数据更新、新消息, 时长: 1-2s周期

### 加载

- 主加载：矩阵雨 + 进度百分比
- 局部：闪烁光标 `▌` 或跳动像素点
- 骨架屏：深灰色块 + 扫描线

## AI 视觉

- **AI标识**: 闪烁终端光标 `▌`，矩阵绿，500ms周期
- **AI气泡**: 背景rgba(0,255,65,0.05)，左边框3px矩阵绿
- **AI思考**: 打字机输出状态文字 + 脑电波/数据流动画
- **用户气泡**: 背景#2A2A2A，右对齐，圆角4px

## 响应式

| 断点 | 范围 | 布局 |
|------|------|------|
| Mobile | < 640px | 底部Tab + 单列 |
| Tablet | 640-1024px | 顶部导航 + 双列 |
| Desktop | > 1024px | 顶部导航 + 多列网格 |

### 尺寸规范

- **TopNavbar**: 移动端: 56px, 桌面端: 64px
- **BottomTab**: 移动端: 64px + 安全区, 桌面端: -
- **Sidebar**: 移动端: 全屏, 桌面端: 260-480px
- **内容最大宽度**: 移动端: 100%, 桌面端: 1400px
- **标准内边距**: 移动端: 16px, 桌面端: 24px

### 间距系统

基础单位: 4px
常用: 4 / 8 / 12 / 16 / 24 / 32px

## Logo 规范

- **类型**: 几何文字标 + 发光
- **字体**: Press Start 2P 或定制几何无衬线
- **结构**: NEO-LOG (可用分隔符: `-` `_` `/`)
- **安全空间**: 四周至少1个"N"字母宽度

**发光效果**:
```css
text-shadow: 0 0 5px #00ff41, 0 0 10px #00ff41, 0 0 20px #00ff41, 0 0 40px #00ff41;
```

**变体**: 全彩发光版(深色背景)、单色版(图标/favicon)、动画版(启动/加载)
