# 项目规划

> NEO-LOG - AI 原生的个人生活追踪系统

## 项目概述

| 属性 | 值 |
|------|-----|
| 名称 | NEO-LOG |
| 定位 | AI 驱动的个人生活追踪与分析系统 |
| 核心价值 | 个性化、智能化、可视化 |
| 目标用户 | 希望追踪生活数据并获得智能洞察的个人用户 |

### 解决的痛点

1. 习惯/目标定义不够个性化，多场景需要多个 App
2. 可视化依赖用户自身分析能力
3. 未利用 AI 能力进行智能分析和建议

## 功能模块

### 1. 登录模块

| 功能 | 说明 |
|------|------|
| 邮箱密码登录 | 支持"记住我"，失败限制 |
| GitHub OAuth | 一键授权登录 |
| 注册流程 | 邮箱验证，密码强度检测 |
| 密码找回 | 6位验证码重置 |
| 访客体验 | 限制功能的演示模式 |
| 会话管理 | 多设备管理，远程登出 |

### 2. Chatbot 模块

| Agent | 触发指令 | 功能 |
|-------|---------|------|
| 行为记录 | `/记录` `/r` | 解析并保存行为记录 |
| 行为查询 | `/查询` `/q` | 时间/类型筛选查询 |
| 目标记录 | `/目标` `/g` | 创建关联行为的目标 |
| 目标查询 | `/查目标` `/qg` | 查看目标进度 |
| 随手记 | `/笔记` `/n` | 快速记录想法 |
| 随手记查询 | `/查笔记` `/qn` | 搜索笔记 |
| 分析 | `/分析` `/a` | 趋势/异常/相关性分析 |
| 前端工具 | `/图表` `/v` | 渲染可视化图表 |

**输入方式**: 文字 / 语音 / 图片

### 3. 用户配置模块

| 配置项 | 内容 |
|--------|------|
| AI配置 | API Key管理、模型选择 |
| 外观 | 主题模式、强调色、动画设置 |
| 通知 | 目标提醒、记录提醒、数据洞察 |
| 数据管理 | 导出、导入、清除 |
| 账户 | 个人信息、密码、设备管理 |
| 语言地区 | 界面语言、时区、货币 |

### 4. 历史记录模块

| 功能 | 说明 |
|------|------|
| 统一时间线 | 行为/目标/随手记混合展示 |
| 视图切换 | 表格 / 卡片 / 时间轴 |
| 搜索筛选 | 全局搜索、类型/时间筛选 |
| 编辑删除 | 侧边栏编辑面板 |

### 5. 可视化模块

| 图表类型 | 用途 |
|---------|------|
| 概览卡片 | 今日统计、目标达成率、连续打卡 |
| 折线图 | 行为趋势 |
| 柱状图/饼图 | 分类对比 |
| 热力图 | GitHub风格活跃度 |
| 进度环 | 目标进度 |

### 6. 表单模块

> AI 不可用时的降级方案

| 表单 | 功能 |
|------|------|
| 行为记录表单 | 动态字段、常用行为快捷选择 |
| 行为定义 | 创建自定义行为类型 |
| 目标表单 | 关联行为、达成条件配置 |
| 随手记表单 | Markdown 编辑、标签管理 |

## 页面结构

| 路由 | 页面 | 权限 |
|------|------|------|
| `/` | 落地页 | 公开 |
| `/login` | 登录 | 公开 |
| `/register` | 注册 | 公开 |
| `/reset-password` | 密码重置 | 公开 |
| `/demo` | 访客体验 | 公开 |
| `/dashboard` | 仪表盘 | 需登录 |
| `/chat` | AI对话 | 需登录 |
| `/history` | 历史记录 | 需登录 |

**侧边栏** (无独立路由): 设置、行为表单、目标表单、随手记表单、记录编辑

## 开发优先级

### P0 - 核心功能

1. 登录/注册 (邮箱密码)
2. Chatbot 基础框架 + 行为记录 Agent
3. 行为定义/记录表单
4. 历史记录 (基础列表)
5. Dashboard (概览卡片)

### P1 - 重要功能

1. 目标模块 (Agent + 表单 + 进度)
2. 随手记模块
3. 行为查询 Agent
4. 可视化图表 (折线图、热力图)
5. 设置页面 (外观、通知)

### P2 - 增强功能

1. 分析 Agent
2. GitHub OAuth
3. 语音/图片输入
4. 数据导出/导入
5. 完整可视化 (对比分析)

### P3 - 优化

1. 访客体验模式
2. 多设备管理
3. 自定义 API 端点
4. PDF 报告导出
5. 性能优化
