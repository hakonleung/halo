# ui-config.md 询问清单

## 问题列表

1. **品牌信息**
   - 设计理念？ → `品牌.理念`
   - Slogan？ → `品牌.Slogan`
   - 视觉风格？(赛博朋克/极简现代/企业商务/活泼多彩) → `品牌.风格`

2. **主题模式**
   - [ ] 仅深色 | [ ] 仅浅色 | [ ] 深色+浅色切换 | [ ] 跟随系统
   - → 影响配色方案

3. **主色调**
   - [ ] 绿色系 (#00FF41) | [ ] 蓝色系 (#00D4FF) | [ ] 紫色系 (#BF00FF) | [ ] 橙色系 (#FF6B35) | [ ] 自定义
   - → `配色.主色` (生成色值表)

4. **字体风格**
   - [ ] 像素/复古 | [ ] 等宽/代码风 | [ ] 无衬线现代 | [ ] 系统默认
   - → `字体` (生成字体配置)

5. **动画强度**
   - [ ] 丰富动效 | [ ] 适度动效 | [ ] 最小动效
   - → `动画` (影响动画配置)

6. **组件库**
   - [ ] Chakra UI | [ ] Shadcn/ui | [ ] Ant Design | [ ] MUI | [ ] 其他
   - → 影响组件规范配置

## 扫描提示 (仅 `/flow-init`)

- UI 库：`package.json` 依赖 (chakra-ui, @radix-ui, antd, @mui)
- 主题配置：`theme.ts`, `tailwind.config.js`
- 颜色/字体：从主题文件提取
