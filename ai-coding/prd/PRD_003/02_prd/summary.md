### 功能范围

**Theme**: 字体、颜色令牌、透明度变体、毛玻璃预设

**核心 Recipe**: Button, Input, Select, Drawer, Card, Badge

**扩展 Recipe**: BottomNav, Popover/Menu, Toast

**动画**: pulse-glow, glitch, scan-line

### 非功能需求

- 动画 FPS ≥ 55
- 字体 LCP < 100ms
- CSS 增量 < 20KB
- WCAG AA 对比度
- reduced-motion 支持
