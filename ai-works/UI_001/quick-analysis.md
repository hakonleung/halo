# 快速分析：字体颜色可见性问题

## 需求理解

**问题**: 字体颜色根本看不见，需要重新定义 theme

**期望**: 所有文字在深色背景上清晰可见，符合 WCAG 对比度标准

**实际**: 当前部分文字颜色在深色背景上对比度不足，特别是 `text.mist` 和 `text.dim`

## 问题定位

### 当前颜色定义

```typescript
text: {
  neon: { value: '#E0E0E0' },  // 荧光白 - 主文字
  mist: { value: '#888888' },   // 灰雾 - 次要文字
  dim: { value: '#555555' },    // 暗淡灰 - 禁用状态
}
```

### 对比度分析

在深色背景 `#0A0A0A` 上的对比度：
- `text.neon` (#E0E0E0): 约 15.8:1 ✓ (优秀)
- `text.mist` (#888888): 约 4.2:1 ⚠️ (接近最低标准 4.5:1，小字体可能不够)
- `text.dim` (#555555): 约 2.6:1 ✗ (严重不足，几乎看不见)

### 使用场景

- `text.neon`: 主文字、标题、重要内容 - 使用广泛，对比度足够
- `text.mist`: 次要文字、标签、占位符 - 使用广泛，对比度不足
- `text.dim`: 禁用状态、辅助说明 - 使用较少，但对比度严重不足

## 修改清单

1. **提高 `text.mist` 亮度**
   - 从 `#888888` 提升到 `#B0B0B0` 或 `#C0C0C0`
   - 确保在深色背景上对比度 ≥ 4.5:1

2. **提高 `text.dim` 亮度**
   - 从 `#555555` 提升到 `#888888` 或 `#999999`
   - 确保在深色背景上对比度 ≥ 3:1 (大文字) 或 ≥ 4.5:1 (正常文字)

3. **保持赛博朋克风格**
   - 确保新颜色仍然符合赛博朋克美学
   - 保持层次感（neon > mist > dim）

## 推荐颜色值

基于 WCAG AA 标准（正常文字 4.5:1，大文字 3:1）：

- `text.neon`: `#E0E0E0` (保持不变，对比度优秀)
- `text.mist`: `#B0B0B0` (从 #888888 提升，对比度约 6.5:1)
- `text.dim`: `#888888` (从 #555555 提升，对比度约 4.2:1，适合大文字或次要信息)

或者更保守的方案：
- `text.neon`: `#E0E0E0` (保持不变)
- `text.mist`: `#C0C0C0` (对比度约 8.5:1，更清晰)
- `text.dim`: `#999999` (对比度约 5.2:1，更清晰)

## 影响范围

- 修改文件: `src/styles/theme.ts`
- 影响组件: 所有使用 `text.mist` 和 `text.dim` 的组件
- 无需修改其他文件（通过 theme token 统一管理）
