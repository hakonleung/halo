# CalendarHeatmap 组件规格

## 概述
GitHub 风格活跃度热力图，展示全年活跃分布。

## Props

```typescript
interface CalendarHeatmapProps {
  data: HeatmapDataPoint[];
  year?: number; // 默认当前年
  onDayClick?: (date: string, count: number) => void;
  loading?: boolean;
}

interface HeatmapDataPoint {
  date: string; // YYYY-MM-DD
  count: number;
}
```

## 状态变体

| 状态 | 样式 |
|------|------|
| Default | 按记录数量显示颜色深浅 |
| Hover | 格子放大 + 发光 + Tooltip |
| Clicked | 打开当日详情弹窗 |
| Loading | 灰色方块 + 闪烁 |
| Empty | 全灰 + 引导文字 |

## 颜色梯度

| 记录数 | 颜色 |
|--------|------|
| 0 | #1a1a1a (bg.carbon) |
| 1-2 | rgba(0, 255, 65, 0.2) |
| 3-5 | rgba(0, 255, 65, 0.4) |
| 6-9 | rgba(0, 255, 65, 0.6) |
| 10+ | rgba(0, 255, 65, 0.9) |

## 尺寸

| 元素 | 尺寸 |
|------|------|
| 单格 | 12px × 12px |
| 间距 | 2px |
| 整体高度 | ~120px |
| 整体宽度 | ~700px (可横滚) |

## 动画

- 首次加载: 格子依次淡入 (stagger 5ms)
- Hover: scale(1.3) + box-shadow 发光
- 点击: 弹出 Modal
