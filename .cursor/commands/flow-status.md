# 查看当前工作流状态

查看当前工作流状态。

## 触发指令

```
/flow-status
```

## 重要提示

**在执行此命令前，必须先读取完整的详细说明文件：**

`ai-coding/action-status.md`

该文件包含完整的执行流程、详细步骤、输出格式等所有必要信息。请先读取该文件，然后按照其中的详细说明执行。

## 执行流程

读取当前工作流的 overview.json，输出状态信息。

## 输出内容

- 当前工作流单号
- 当前阶段
- 阶段状态（pending/in_progress/completed）
- 风险等级
- 人类介入要求

## 参数说明

无参数

## 详细说明

完整的工作流说明、执行步骤、输出格式等详细信息，请参考：

`ai-coding/action-status.md`

**请务必在执行前阅读该文件的完整内容。**
