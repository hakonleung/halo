# 轻量工作流

快速迭代模式，跳过产品设计阶段，直接进入技术实现。

## 触发指令

```
/flow-quick [需求描述]
```

## 适用场景

- Bug 修复
- 小功能迭代
- 技术优化
- UI 微调

## 执行流程

1. **Q1 快速分析** - 理解需求，定位代码，生成修改清单
2. **Q2 直接实现** - 按清单修改，验证编译
3. **Q3 验证** - 功能验证、编译检查、Lint检查、单元测试

## 何时升级到完整模式

- 修改文件 > 10
- 需要新增数据表
- 需要新增 API > 3
- 需要新增页面
