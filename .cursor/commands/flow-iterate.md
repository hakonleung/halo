# 迭代工作流

在已完成的 PRD 基础上进行功能迭代，复用已有设计文档。

## 触发指令

```
/flow-iterate [PRD_XXX] [迭代需求]
```

## 适用场景

- 功能增强
- 需求变更
- 版本迭代
- 补充功能

## 执行流程

1. **I1 增量需求分析** - 加载原 PRD 上下文，对比分析
2. **I2 增量设计** - 生成补丁文档（PRD/UI/技术）
3. **I3 增量实现** - 按补丁文档修改代码
4. **I4 回归验证** - 增量测试 + 回归测试
